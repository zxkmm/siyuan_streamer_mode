# 什么是主播模式
Streamer Mode 是一些软件（尤其是游戏和直播软件）提供的一种特殊模式，用于在直播或录制视频时保护用户的隐私和安全。开启 Streamer Mode 后，软件会屏蔽某些敏感信息，例如：

1. **个人信息**：用户名、电子邮件地址、真实姓名等。
2. **密码**：屏蔽你定义的敏感的密码。
3. **IP地址**：屏蔽你定义的敏感的IP。
4. **链接**：屏蔽你定义的敏感的链接。

这样可以让主播在直播过程中不必担心隐私泄露或受到骚扰，提高直播的安全性和隐私保护。
